# Photos du catalogue formations

Ce document recense les images réalistes générées et branchées pour les cartes formations LODENE. Les assets finaux sont en WebP, ratio horizontal proche 16:9, dimensions `1672x941`, dans `public/formations/photos/`.

## Direction commune

- Rendu photoréaliste, premium, lumineux et professionnel.
- Personnes en situation réelle avec matériel crédible.
- Palette LODENE : turquoise / bleu canard / blanc / gris.
- Branding discret par EPI, véhicules, documents ou accents turquoise.
- Pas d'illustration vectorielle, pas d'icône en image principale, pas de doublon entre slugs.
